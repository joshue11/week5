Website Planning Document

Site Name

HealthyLife Recipes Hub

I chose this name because the website will focus on providing simple, healthy, and beginner-friendly recipes for people trying to improve their eating habits.

Optional domain availability: healthylife-recipes.org

Site Purpose

This website provides a collection of healthy recipes, meal preparation tips, nutrition information, and cooking guides for beginners. Users will be able to browse recipes by category and learn how to cook healthier meals at home.

Scenarios

  • What healthy meals can I cook quickly after work?
  • Where can I find beginner-friendly recipes with simple ingredients?

Color Schema

Selected Colors:

  • #2a5d94 – Primary color used for headings and accents.
  • #d98324 – Accent color used for highlights and important text.
  • #f4f7fa – Light background color for readability.

This site plan document uses these exact colors for headings, accents, and background.

Typography

  • Roboto – Used for all headings and body text.

This site plan document uses the Roboto font throughout.
