MyFitnessJourney – Website Planning Document

Site Name

MyFitnessJourney

Escogí este nombre porque representa un sitio web dedicado a ayudar a las personas a mejorar su salud y crear una rutina personal de ejercicios.

Optional Domain: myfitnessjourney.fit

Site Purpose

El propósito del sitio es proporcionar recomendaciones de ejercicio, rutinas personalizadas, información nutricional y una forma de seguimiento del progreso físico.

Scenarios

  • ¿Qué ejercicios puedo hacer para bajar de peso desde casa?
  • ¿Dónde puedo ver una rutina recomendada para principiantes?

Color Schema

Primary Color – #0C5FA8 (Se usará para encabezados, botones y acentos importantes)

Secondary Color – #F2F7F9 (Se usará como color de fondo)

Typography

Poppins – Se usará para todo el sitio, incluyendo encabezados y texto del cuerpo por su claridad y estilo moderno.
